Weather App Project, from The Odin Project - Completed

Details:

*Created a simple but effective weather app which displays the weather conditions and current temperature of whichever city the user searches.
*Used asynchronous API calls to achieve this goal.
*Implemented weather icons that match the gathered conditions data from the API call.
*Implemented the ability to switch temperature between celsius and fahrenheit by storing the already returned data and converting it (no second API call necessary).
